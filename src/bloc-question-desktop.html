LA QUESTION DU MOIS
QUAND DOIS-JE UTILISER LE LOGO PMU EN FORMAT POP-UP?
C’est simple : tout le temps, ou presque! Il n’y a que deux cas dans lesquels on ne l’utilise pas:
  -  Pour les communications sponsoring.
  -  Quand PMU est partenaire d’un événement lié à la filière.
Sur toutes les autres prises de parole, le format pop-up reste le logo de référence
